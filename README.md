##
Папка findingToken, в ней находится код-тест для токена длины в 7(UUID в 4.5 раз больше) символов и алфавитом в 16(как и в UUID)
На моем ноутбуке требовалось по 10 секунд для поиска. Не нужно быть гением арифметики, чтобы понять, что при добавлении каждого символа в длину, время в среднем будет увеличиваться в 16 раз. Итого, даже если существует около миллиона токенов, по моим подсчетам, выйдет, что потребуется на поиск 10с*(16^25)/1000, этоприблизительно 400.000.000.000.000 лет
