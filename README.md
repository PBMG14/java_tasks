# java_tasks
В данном репозитории:
Папка enhorse, там весь код, что был в той статье.
Папка findingToken, в ней находится код-тест для токена длины в 7(UUID в 4.5 раз больше) символов и алфавитом в 16(как и в UUID)
Папка java_1, в ней находится два скрипта: один - подключение к Яндекс погоде, затем взятие отттуда json и последующее его дробение для показа в нормальном формате
                                           второй - локальный сервер, в качестве удобства и наглядности на странице login имеются ссылки ведущие на остальные страницы, в них используется токен. Конечно, в реальности он не должен быть открытым, однако тогда и работу наглядно не показать. delete работает только через curl.
Также добавил работу по базам данных: там сделано подключение через jdbc к postgreSQL, создано клиентское приложение с интерфейсом. Что-то нейронка писала, что-то я.
